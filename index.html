Projects

Here I have listed some of my cool projects

Sobel Edge Detection

Implemented Sobel edge detection on an ASIC. This algorithm is often used in the fields of computer vision and image recognition to detect the edges of images. ASIC was designed in Verilog and mapped version of design was tested appropriately with test benches. The Sobel filter involves matrix manipulation and we used pipeling to optimize the edge detection steps. Furthermore, the output image is a black-white representation of the original image with the white borders indicating the edges.

Huffman Coding

This project highlights a lossless data compression algorithm that was implemented in C. This algorithm works on the principle that a file containing maximum number of repeated characters is compressed the most. The technique works by creating a binary tree of nodes. The picture towards the right explains an example of Huffman Coding and bit manipulation done.

We were awarded the second place in the Nationwide Engineering Academic Program Contest of 2016. This design project involved using the HCS12 Microcontroller and Arduino Nano.
